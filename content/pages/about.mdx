# 🏭 MxCore — Industrial Knowledge Base & Blog Platform

**MxCore** adalah proyek **open-source** berbasis **Next.js (App Router)** yang dirancang untuk menjadi **platform blog dan dokumentasi teknis** khususnya di bidang **industri, petrochemical, proyek engineering, dan safety**.

Dengan dukungan struktur konten modular menggunakan **Contentlayer**, MxCore menyederhanakan penulisan, manajemen, dan distribusi artikel dalam format **Markdown (MDX)**. Didukung oleh desain minimalis dan performa tinggi, MxCore sangat cocok untuk **insinyur, profesional industri, dan tim teknis**.

---

## 🎯 Tujuan Proyek

MxCore dibuat untuk:

- ✍️ Mendokumentasikan pengetahuan industri: maintenance, safety, EPC, proyek, dsb
- 📚 Menyusun blog dan pusat dokumentasi teknis berbasis Markdown
- 🧠 Membangun _knowledge base_ untuk tim teknis / engineering
- 🛠️ Meningkatkan efisiensi manajemen konten tanpa backend atau CMS
- 🌐 Menyediakan platform statis untuk **personal branding** dan publikasi

## 📚 Struktur Konten

Struktur folder konten disesuaikan langsung dari direktori:

```

content/
├── author/
├── blog/
│   ├── AI/
│   ├── design/
│   ├── maintenance/
│   ├── project/
│   └── safety/

```

Setiap file `.mdx` memiliki metadata `frontmatter`, seperti:

```md
---
title: TOR Reverse Engineer STHE
tags: ['project', 'tor', 'design']
date: '2025-10-10'
draft: false
---
```

## 🛠️ Cara Mengembangkan

1. Clone repository:

   ```bash
   git clone https://github.com/namamu/mx-core.git
   cd mx-core
   ```

2. Install dependencies:

   ```bash
   npm install
   ```

3. Jalankan local development:

   ```bash
   npm run dev
   ```

4. Build dan export (statis):

   ```bash
   npm run build && npm run export
   ```
