Dalam perhitungan load bearing, terdapat 2 jenis beban yang perlu dihitung, yakni beban radial, yang arahnya tegak lurus terhadap shaft dan beban axial, yang arahnya sejajar dengan orientasi shaft

1 Perhitungan radial load bearing
∑M=0
(Untuk Bearing Drive End)
Perhitungan parameter didasarkan atas beban dan jarak beban terhadap masing-masing bearing

    ∑Fy=0
    (Untuk Bearing Non-Drive End)

2 Perhitungan Axial Load Bearing
Fa = (Fr)/(2 x (Y))

3 Load Calculation for Each Bearing
Fe = X.Fr+Y.Fa - Parameter X dan Y mengacu pada konstanta yang ditetapkan oleh pihak manufaktur dan tercantum pada katalog/spesifikasi bearing

Calculation for Single/Tandem angular contact ball bearing

![gambar5](/static/images/artikel/Gambar-5-Calculation-Factor-for-Single-Tandem-Bearing.jpg)

Source : https://www.skf.com/sg/products/rolling-bearings/ball-bearings/angular-contact-ball-bearings/single-row-angular-contact-ball-bearings/productid-7311%20BECBM

Given :
X = 0,35
Y = 0,57

Load Calculation for DE Bearing with single configuration
Fe = X.Fr+Y.Fa
= (0,35) (0,9) + (0,57) (0,8)
= 0,315 + 0,456
= 0,771 kN

Load Calculation for NDE Bearing with single configuration
Fe = X.Fr+Y.Fa
= (0,35) (1,5) + (0,57) (1,3)
= 0,525 + 0,741
= 1,266 kN

Calculation for Double Pair Bearing

![gambar6](/static/images/artikel/Gambar-6-Calculation-Factor-for-Double-Pair-Bearing.jpg)

Source : https://www.skf.com/sg/products/rolling-bearings/ball-bearings/angular-contact-ball-bearings/single-row-angular-contact-ball-bearings/productid-7311%20BECBM

Given :
X = 0,57
Y = 0,93

Load Calculation for DE Bearing with double pair configuration
Fe = X.Fr+Y.Fa
= (0,57) (0,9) + (0,93) (0,8)
= 0,513 + 0,744
= 1,257 kN

Load Calculation for NDE Bearing with double pair configuration
Fe = X.Fr+Y.Fa
= (0,57) (1,5) + (0,93) (1,3)
= 0,855 + 1,209
= 2,064 kN
