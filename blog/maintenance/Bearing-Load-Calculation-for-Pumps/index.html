Bearing Load Calculation for Centrifugal Pumps

Authors
  • Name
    Faisal Andhika

Dalam perhitungan load bearing, terdapat 2 jenis beban yang perlu dihitung, yakni beban radial, yang arahnya tegak lurus terhadap shaft dan beban axial, yang arahnya sejajar dengan orientasi shaft

1 Perhitungan radial load bearing ∑M=0 (Untuk Bearing Drive End) Perhitungan parameter didasarkan atas beban dan jarak beban terhadap masing-masing bearing

∑Fy=0 (Untuk Bearing Non-Drive End)

2 Perhitungan Axial Load Bearing Fa = (Fr)/(2 x (Y))

3 Load Calculation for Each Bearing Fe = X.Fr+Y.Fa - Parameter X dan Y mengacu pada konstanta yang ditetapkan oleh pihak manufaktur dan tercantum pada katalog/spesifikasi bearing

Perhitungan beban yang bekerja pada bearing

∑M=0 [(300 x 500)+(200 x 500)]-[(Fr A x 250)+(100 x 250)]=0 [150000+100000]= [(Fr A x 250)+25000] Fr A=900 N (0,9 kN) (Radial Load for DE Bearing)

∑Fy=0 Fr B=900N+300N+300 N Fr B=1500N (1,5kN) (Radial Load for NDE Bearing)

Source : https://koyo.jtekt.co.jp/en/support/bearing-knowledge/5-4000.html

Fa A = (0,9)/(2 x (0,57)) Fa A = (0,9)/(1,14) Fa A = 0,8 kN (Axial Load for DE Bearing)

Fa B = (1,5)/(2 x (0,57)) Fa B = (1,5)/(1,14) Fa B = 1,3 kN (Axial Load for NDE Bearing)

Calculation for Single/Tandem angular contact ball bearing

Source : https://www.skf.com/sg/products/rolling-bearings/ball-bearings/angular-contact-ball-bearings/single-row-angular-contact-ball-bearings/productid-7311%20BECBM

Given : X = 0,35 Y = 0,57

Load Calculation for DE Bearing with single configuration Fe = X.Fr+Y.Fa = (0,35) (0,9) + (0,57) (0,8) = 0,315 + 0,456 = 0,771 kN

Load Calculation for NDE Bearing with single configuration Fe = X.Fr+Y.Fa = (0,35) (1,5) + (0,57) (1,3) = 0,525 + 0,741 = 1,266 kN

Calculation for Double Pair Bearing

Source : https://www.skf.com/sg/products/rolling-bearings/ball-bearings/angular-contact-ball-bearings/single-row-angular-contact-ball-bearings/productid-7311%20BECBM

Given : X = 0,57 Y = 0,93

Load Calculation for DE Bearing with double pair configuration Fe = X.Fr+Y.Fa = (0,57) (0,9) + (0,93) (0,8) = 0,513 + 0,744 = 1,257 kN

Load Calculation for NDE Bearing with double pair configuration Fe = X.Fr+Y.Fa = (0,57) (1,5) + (0,93) (1,3) = 0,855 + 1,209 = 2,064 kN
